UDP server installation for ZIVPN Tunnel (SSH/DNS) VPN app.
<br>

>Server binary for Linux amd64 and arm.

#### Instalation Menu

```
wget -O install.sh https://raw.githubusercontent.com/altunnel/udp-zivpn/main/install.sh
chmod +x install.sh
./install.sh
```
